# T-tube 居家照護

## 一、 定義

當病人行膽道相關手術後，因手術後可能會發生膽道水腫，為了防止膽汁滲入腹膜腔及維持膽道通暢，在總膽管處放置T型引流管，將膽汁引流到體外。T型管留置的時間，視個人病況不同而異，醫師會依據膽管攝影及膽道鏡檢查之結果決定，通常需留置數週至數月之久。

二、適用對象

1. 總膽管造口術病人。

2. 膽囊切除。

3. 其他膽道外科手術病人。

## 三、 T型管的傷口護理

1. 先用肥皂洗净雙手。

2. 每天以生理食鹽水或冷開水或冷開水沾濕棉棒，由內而外擦拭 T 型管周圍的傷口。

3. 乾燥後使用優碘藥水消毒，並覆蓋 Y 型紗布，再以膠布固定。

4. 每天至少更換紗布一次，依傷口分泌物多寡，而增減換藥的次數。



5. 管子周圍保持清潔、乾燥，以避免刺激傷口及皮膚。

6. 引流管先做出抛物線狀態再以膠布黏貼成

(#)字型固定於腹部，以防因牵扯而滑脱，每天應更换黏貼的位置，避免因膠布造成皮膚過敏，且固定時應小心，勿造成T型管轉折不通。

7. 引流管收集袋不可高過於引流管，避免引流液回流造成汙染，收集袋可適度固定在大腿或腰際間。

四、T型管照顧的注意事項

1. 記錄每日的引流量及顏色。

2. 保持皮膚清潔。

3. 定期更換敷料。

4. 定期更換引流袋。

5. 避免劇烈及過度的活動（如搬重物、游泳、跑步等），以避免膽汁流出太多，活動時，應固定在腰部或腿部。

6. 若膽汁沾到而弄髒衣服時，可以用漂白水或蘇打水浸泡衣服後再洗淨。

7. 有下列情形出現，請至急診處理：



(1)高燒不退。

(2)腹痛、劇烈嘔吐。

(3) 引流管阻塞不通。

(4) 引流管脱出。

(5)引流管的伤口，红腫壓痛或有膿狀分泌物流出。

(6) 引流液的颜色呈现红色或引流量减少伴有腹胀、腹痛。

## 五、 饮食方面：

1. 定時、定量，細嚼慢嚥，而且飯後宜休息30分鐘。

2. 避免食用油腻、油炸、糯米類以及易產氣食物（如大豆、洋蔥、甘蕃等）。

3. 可多喝運動飲料補充體內電解質。

4. 應充分休息，避免過度勞累。

5. 盡量穿著寬鬆衣服，以避免壓迫引流管。

六、關閉T型管注意事項（於醫師告知後才開始執行）

1. 開始時每天關閉三至四小時，再打開十五分鐘（依醫師指示）。

2. 晚上不必關閉。3. 如腹部有饱胀感，可以打開引流，再予以關閉。

4. 二至三天後，如無不適現象，則白天、夜晚全部關閉不必打開。
