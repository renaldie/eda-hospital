# 經皮內視鏡胃造廬 術後之照護注意事項

## 一、 名詞解釋

病人可能因某疾病發生吞嚥困難而需要管灌飲食，胃造壞為一個替代鼻胃管的方式，此為在胃及腹壁上做一造口，置入灌食管。

## 二、 通用對象

1.腦中風患者。

2.頭頸部癌或食道癌治療後吞嚥困難。

3.腦部病變或神經肌肉系統疾病而需長期置放鼻胃管者。

4. 無法放置鼻胃管之患者。
