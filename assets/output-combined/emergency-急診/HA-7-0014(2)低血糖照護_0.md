# 低血糖照護

## 什麼是低血糖？

低血糖通常是指血糖濃度低於70mg/dl以下，就會容易產生所謂低血糖的症狀，所以當血糖低於70mg/dl時，就應注意低血糖發生的可能性。

## 為什麼會發生低血糖？

1. 注射或服藥後未按時間進餐

2. 口服藥或胰島素使用過量

3. 突然運動量增加，未補充適當的點心

4. 空腹過久未進食
