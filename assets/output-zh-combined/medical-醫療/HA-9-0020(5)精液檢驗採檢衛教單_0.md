## 精液檢驗 採檢衛教單

一、檢驗項目：精液檢驗(Semen analysis)

二、採檢前注意事項：採檢前應禁慾2-7天。

## 三、採檢步驟：

1、以手淫方式，勿使用保險套，將一次全部精液量收集至40ml量杯中。

2、填寫回覆單：

| 姓名： | 出生年月日： |
|--------|--------------|
| 檢體是否為已結紮檢體：☐ 已結紮；☐ 未結紮 |  |
| 禁慾天數：☐<2 天；☐2-7 天；☐>7 天 |  |
| 採檢方式：☐手淫；☐其他(請註明)： |  |
| 採檢量：☐一次全部的精液量；☐有遺漏約 ___ ml |  |
| 採檢時間：___月 ___日 ___時 ___分 |  |

## 四、採檢後注意事項：

1、檢體室溫保存，不可冷藏或加溫，並於30分鐘內將檢體與回覆單繳交至檢驗部/科收件窗口。
