| 第17天 | ☑取精 | · 取卵日上午請先生取精。取精後，存放於取精罐中。若取卵當天先生無法配合取精，可事先將精子取出冷凍。冷凍精子和新鮮精子並不會影響胚胎品質和試管嬰兒成功機率。 | 精蟲冷凍費用及保存費用另計 |
|  | 取卵 | · 取卵手術採靜脈注射全身麻醉，因此前一晚12點過後，須禁食（含水）。· 上午進行取卵手術，手術結束後，將在恢復室觀察30-60分鐘。· 待麻醉科醫師評估後，抽完紗布、解過小便後，便可以出院返家，不需要住院。 |  |
|  | 體外受精 | · 中午於實驗室進行受精。· 受精可分成體外自然受精(IVF)及單一精蟲顯微注射(ICSI)兩種。一般採體外自然受精，特殊情況將採單一精蟲顯微注射。· 單一精蟲顯微注射費用係以≤5顆計算，之後每增加1顆，再加收費用。 | 單一精蟲顯微注射費用另計 |
| 第18-22天 | 胚胎培養 | · 順利的話，第3天一個胚胎會有8顆細胞。 |  |
| 第20-22天 | 新鮮胚胎植入或胚胎冷凍 | · 胚胎培養可於第3天或第5天植入，或選擇胚胎培養3-5天冷凍保存。· 目前人工生殖法規定胚胎冷凍時間最長為10年。 | 胚胎冷凍費及培養費用另計 |
| 第22-23天 | 胚胎著床前遺傳診斷(PGS/PGD) | · 若因醫療目的而需要接受著床前遺傳診斷(PGD)或著床前染色體篩檢(PGS)者，於取卵手術後5天進行，胚胎植入將順延一個週期。 | 著床前遺傳診斷費另計(實際需求依病人情況選擇) |
