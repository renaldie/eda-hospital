## E-DA HEALTHCARE GROUP

貳、

<div style="text-align: center;">冷凍胚胎解凍植入治療項目</div>

| 時間 | 流程 | 說明 | 備註 |
|------|------|------|------|
| 月經週期第21天 | 打調荷爾蒙針 | 使內在荷爾蒙休息<br>• 打完針劑7-10天月經會來。<br>• 少數人打完針隔14天月經沒來,須回診催經。 |  |
| (下一次)月經週期第2天 | 口服雌激素 | 刺激內膜增生<br>• 連續服用藥物12天(服用藥物期間勿漏或忘記服用,當有此情形下餐服藥時必須把漏服用的藥物補上) |  |
| 第13天 | • 續口服雌激素(E2)<br>• 照陰道超音波 | 內膜厚度及形狀以達最佳植入時間<br>• 內膜厚度達≥0.8cm適合胚胎著床。<br>• 醫師會與個案討論決定植入時胚胎等級。<br>• 依照胚胎培養天數決定何時植入。 |  |
| 第13天起 | 補充黃體素 | 加強黃體功能<br>• 為穩定子宮內膜,胚胎植入後,須遵照醫囑補充黃體素約14-16天,以利胚胎著床。 |  |
| 第16-18天 | 胚胎植入 | • 植入當天早上才會把胚胎解凍。<br>• 胚胎植入的過程不需要麻醉,植入後也不會感到不適。需要平躺2小時。 |  |
| 第23-25天 | 回診 | 清理黃體素陰道凝膠<br>• 清理黃體素陰道凝膠。 |  |
| 月經週期第30-32天 | 驗孕 | 確認懷孕與否<br>• 植入2週後驗孕。 |  |
| 第37-39天 | 回診 | 確認胚胎著床處正確與否 |  |
| 第44-46天 | 回診 | 確認胚胎是否有心跳 |  |
