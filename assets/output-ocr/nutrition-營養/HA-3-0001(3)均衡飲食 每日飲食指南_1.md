## 均衡飲食

一般原則

每日應從各種食物中均衡的攝取各類營養素，以提供身體所需，並養成不偏食、不暴飲暴食的習慣。

第一類 乳品類（1杯 240毫升）

【我的餐盤口訣：每天早晚一杯奶】

## 第二類 水果類(每天：2-4份)

【我的餐盤口訣：每餐水果拳頭大】

<table border=1 style='margin: auto; width: max-content;'><tr><td style='text-align: center;'>營養成分</td><td style='text-align: center;'>醣類、膳食纖維、維生素及植化素。</td></tr><tr><td style='text-align: center;'>食物來源</td><td style='text-align: center;'>各種新鮮水果。</td></tr><tr><td style='text-align: center;'>小提醒</td><td style='text-align: center;'>含有醣份須適量攝取，果乾和果汁含糖 量較高，且易過量，建議以攝取新鮮水 果為主，可挑選當令水果，多樣化攝取。</td></tr></table>

【我的餐盤口訣：菜比水果多一點】

<table border=1 style='margin: auto; width: max-content;'><tr><td style='text-align: center;'>營養成分</td><td style='text-align: center;'>膳食纖維、維生素、植化素與礦物質。</td></tr><tr><td style='text-align: center;'>食物來源</td><td style='text-align: center;'>葉菜類、花菜類、根菜類、豆菜類、菇類、海菜類等。</td></tr><tr><td style='text-align: center;'>小提醒</td><td style='text-align: center;'>高膳食纖維、熱量低又可增加飽足感，並刺激腸道蠕動。醃漬醬菜類鈉含量較高，且添加物較多，以新鮮或冷凍蔬菜為佳。</td></tr></table>

第五類 豆魚蛋肉類(每天：3-8份)【我的餐鰞口訣：豆魚蛋肉一掌心】


<table border=1 style='margin: auto; width: max-content;'><tr><td style='text-align: center;'>營養成分</td><td style='text-align: center;'>蛋白質與脂肪。</td></tr><tr><td style='text-align: center;'>食物來源</td><td style='text-align: center;'>豆腐、豆干、豆漿、魚類、海鮮、蛋類、雞、鴨、牛、羊、豬...等。</td></tr><tr><td style='text-align: center;'>小提醒</td><td style='text-align: center;'>植物性蛋白質含膳食纖維，飽和脂肪較低且不含膽固醇，為最佳選擇，其次挑選白肉，適度攝取紅肉。去除肥皮，蒸、烤、燙、滷、燉煮等少油烹調方式為佳。</td></tr></table>

第六類 油脂與堅果種子類（油脂3-7茶匙及堅果種子類1份）【我的餐盤口訣：堅果種子一茶匙】


<table border=1 style='margin: auto; width: max-content;'><tr><td style='text-align: center;'>營養成分</td><td style='text-align: center;'>堅果種子提供不飽和脂肪酸和維生素E、維生素B1、鉀、鎂、磷、鐵等。</td></tr><tr><td style='text-align: center;'>食物來源</td><td style='text-align: center;'>沙拉油、玉米油、花生油、橄欖油等。</td></tr><tr><td style='text-align: center;'>小提醒</td><td style='text-align: center;'>選擇植物性油脂，避免動物性油脂。每餐攝取1茶匙堅果種子（約大拇指第一指節量）。油脂熱量密度高，應酌量攝取。</td></tr></table>
