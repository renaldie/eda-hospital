| 类别 | 可食用 | 不可食用 |
|------|--------|----------|
| 乳制品及其制品 | 无 | 乳制品（牛奶、乳酪、酸奶、优酪乳、乳清）及含奶的巧克力、布丁、奶油、冰淇淋等 |
| 全谷杂粮类 | 米类、面粉、未加盐的馒头/米粉/冬粉、谷类脆片及纯燕麦片、去皮马铃薯、去皮地瓜、芋头、玉米、薏仁、莲子、山药、栗子、菱角、莲藕 | 烘焙食品：面包、饼干、吐司、糕点等、面条、面线、马铃薯皮、地瓜皮、皇帝豆、青豆仁、花豆 |
| 肉类 | 新鲜未加工肉类（不超过4两/天，1两=37.5g） | 肉松、肉干、培根、香肠、火腿、肉制品罐头等加工制品、内脏 |
| 蛋类 | 蛋白 | 蛋黄及其制品（蛋糕、美奶滋等） |
| 黄豆类及制品 | 无 | 黄豆及其制品（豆腐、豆干、豆浆、酱油等） |
| 水产类 | 无 | 所有水产品（鱼、贝、虾、蟹） |
| 油脂类 | 植物油、无调味坚果（粉）、无盐花生酱、无盐人造奶油 | 加盐坚果类、加盐花生酱、加盐人造奶油 |
| 水果类 | 新鲜水果及果汁 | 果干、蜜饯、含红色色素的水果罐头、含红色色素的果汁 |
| 盐及调味料 | 无碘盐、白醋、无盐调味料 | 海盐、加碘盐、乌醋、酱油、番茄酱、鱼露、加盐的调味料（如：胡椒盐）、昆布类浓缩调味料、料理米酒 |
| 其他 | 糖、蜂蜜、不含红色色素的果酱、现泡黑咖啡、现泡茶、汽水、纯黑巧克力、纯可可粉 | 含碘维生素与营养补充品、含红色色素的糖果与果冻、含海洋原料的营养补充品（鲨鱼软骨素、甲壳素） |

参考资料：Low-Iodine Cookbook. Guidelines, Tips, & Recipes for the Low-Iodine Diet Used Prior to Radioactive Iodine Treatment. 2022, 9th Edition.
