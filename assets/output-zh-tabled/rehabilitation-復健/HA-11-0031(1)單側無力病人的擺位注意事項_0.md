| **單側無力病人的擺位注意事項～躺姿～** |  |  |
| **目的**：1. 避免形成壓瘡、關節攣縮等不良姿勢。2. 維持舒適姿勢。 | 以右側偏癱為例 |  |
| **注意事項**：1. 每兩個小時變換姿勢，並檢查受壓側的皮膚是否出現壓瘡。2. 若有水腦、或骨折處，需依醫師指示調整姿勢。3. 使用呼吸器、尿管者，請留意勿拉扯。 |  |  |
| **仰 躺** | 1. 頭頸部：用枕頭墊高<br>2. 患側手：下方放置枕頭手肘伸直，手心向上/下<br>3. 患側腳：放在正中位置<br>大腿外側放毛巾捲(避免過度外轉)<br>膝下方稍微墊高(避免膝過度伸直)<br>腳踝垂直，不歪斜 | ![img](imgs/img_in_image_box_638_400_941_553.jpg) | ![img](imgs/img_in_image_box_964_399_1151_553.jpg) | ![img](imgs/img_in_image_box_626_563_1165_754.jpg) |  |
| **側躺～患側在上～** | 1. 頭頸部：用枕頭墊高<br>2. 患側手：放在身體前方<br>下方用枕頭或棉被墊高<br>手肘微彎，手腕伸直，手心向下<br>3. 患側腳：向前彎曲<br>髋、膝、踝下方放置棉被等支撐物 | ![img](imgs/img_in_image_box_618_845_1164_1127.jpg) |  |  |
| **側躺～患側在下～** | 1. 頭頸部：用枕頭墊高<br>2. 軀 幹：背、臀後方有支撐，身體微向後<br>3. 患 側：手放在身體前方<br>肩胛骨稍微往前拉，手心向上<br>腳微彎<br>4. 健側腳：向前彎曲<br>髋、膝、踝下方放置棉被等支撐物 | ![img](imgs/img_in_image_box_615_1212_1168_1554.jpg) |  |  |
