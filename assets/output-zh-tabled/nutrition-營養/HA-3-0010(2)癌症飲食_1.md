| 症狀 | 飲食原則及改善方法 |
|------|------------------|
| A.食慾不振 | ·少量多餐、選高熱量高蛋白食物<br>·經常變化烹調方式與型態<br>·選擇喜愛的食物並放鬆心情享用 |
| B.味覺改變 | ·利用辛香料降低肉的苦味<br>·補充含餅食物：海鮮（蚵仔/蛤蠣）、雞蛋等，以改善味覺 |
| C.口乾 | ·含冰塊、喝開水/無糖飲品<br>·選擇濕潤食物（蒸蛋/奶昔等）<br>·避免辛辣食物、飲酒 |
| D.噁心/嘔吐 | ·飯後適度休息、少量多餐<br>·進食時遠離油煙、異味處 |
| E.腹瀉/腹脹 | ·注意水份、電解質補充<br>·避免易產氣、粗糙食物，如洋蔥、豆製品、地瓜等 |
| F.口腔潰瘍 | ·避免刺激性（辛辣、酸性）食物<br>·選擇流質食物或市售營養品<br>·適量補充麩醬酸10-30克/天 |
