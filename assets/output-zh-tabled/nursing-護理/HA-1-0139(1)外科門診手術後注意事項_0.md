# 外科門診手術後注意事項

## ___ 先生/女士

## 一、 藥物使用：

☐1. 口服藥請醫藥袋註明之劑量、次數及服用時間服用；請勿過量服用。

☐2. 外用藥物也請按指示之使用方式外用，切勿口服。

二、傷口處理方法：





☐1. 可吸收縫線縫合的傷口（以透氣膠布貼合者），只需每日替換覆蓋的紗布；所貼合的3M膠布請勿更換或撕掉，傷口亦不需另外敷藥。

☐2. 使用不可吸收縫線縫合的傷口（未用透氣膠布貼合者，需拆線者），請依外用藥物使用方式換藥即可。

☐3.傷口回診前請保持乾燥勿沾水、浸水以避免傷口感染。

4. 若傷口紅、腫、熱、痛超過2天以上或有不正常之分泌物請馬上至門診或急診追蹤並檢視傷口。

□5.若有引流管之傷口，請每日以優碘擦式傷口；覆蓋引流管的乾紗布，請於：最外層還可見血漬或分泌物時馬上更換，若無則每日更換一次即可。

☐6.若傷口持續流血，經直接壓迫15分鐘仍未止血時。

未離院，請回手術室檢查；已離院，請回急診檢查。

☐7.用於固定及壓迫傷口的彈性紗布或繃帶，若遠端肢體有發白、

冰冷、發麻或明顯腫脹情形時，請重新解開彈紗或彈繃，再重新輕輕纏繞傷口

即可；若10分鐘後情形仍未改善請至急診檢查。
