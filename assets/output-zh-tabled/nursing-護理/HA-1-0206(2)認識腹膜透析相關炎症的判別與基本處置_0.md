# 認識腹膜透析相關炎症的判別與基本處置 (腹膜炎、導管出口處發炎、隧道發炎)

## 一、 腹膜炎

(一)症状：

1. 透析液混濁。

2. 腹痛。

3. 發燒、畏寒。

4. 胃肠道症状：噁心、嘔吐、腹瀉、便秘。

5. 引流液有纖維蛋白。

6. 腹膜透析脱水變差。

7. 導管引流不順。

## (二) 基本處置：

1. 當發現透析液混濁時，可能發生了腹膜炎。

2. 請立即聯繫腹膜透析室護理師，說明狀況及處理方式，並到院檢查及治療。

3. 週一~週五 17:00 後及週六、週日發現引流液混濁，請攜帯平常會使用的透析液及換藥水會用到的口罩、磅秤、擦手紙、75%酒精到急診就診。

4. 來醫院前，確認肚子內有注入透析液，2小時後才能引流出來檢查是否有細菌感染；來醫院後將腹內透析液引流出來，交給護理師採檢。

5. 透析液檢查報告出來後，如果細菌量超過標準值，則是腹膜炎，醫師會開立抗生素使用。

6. 如果沒有發燒及嚴重腹痛症狀，經過腎臟科醫師診療同意後，可在家自行注射抗生素於藥水袋內治療，經由腹膜透析護理師教導加抗生素於藥水袋的技術後，即可回家自行加藥治療。

7. 要注意！如果回家後 3 天透析引流液依然混濁，或有腹痛及水腫等狀況，請立即聯絡腹膜透析室。

## 二、 導管出口處發炎：

（一）異常的導管出口：有以下狀況發生，請立即聯絡腹膜透析室護理師。

1. 傷口紅、腫、熱、痛、分泌物。

2. 袖釦滑脫頂在出口處。

3. 毛氈袖釦滑脫的傷口。

4. 傷口長瘋肉，不穩定的傷口。
