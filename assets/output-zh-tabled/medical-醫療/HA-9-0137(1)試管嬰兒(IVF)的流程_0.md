# 試管嬰兒療程說明

人工生殖法規定已婚夫妻始能接受人工生殖技術，欲接受治療者，請確認已完成結婚登記。壹、

試管嬰兒係指卵子由母體取出，讓精卵在體外結合後，經過體外培養3-5天，再行植回母體的方式。

## （一） 檢查項目

| 時間 | 流程 | 說明 | 備註 |
|------|------|------|------|
|  | ○精蟲分析 | 確認精蟲狀況 |  |
| 月經週期第2-3天 | 卵巢荷爾蒙檢驗 | 確認卵巢功能• 抽血驗E2、LH、FSH、TSH、AMH、Prolactine等。 |  |
| 經期結束且乾淨後第3-5天 | 子宮鏡檢查(醫師視情況安排檢查) | 確認子宮狀況• 子宮鏡檢查的過程不需要麻醉。• 子宮鏡檢查發現異常者，需另安排手術。手術採全身麻醉，需要住院3天2夜。 | 手術、麻醉及住院費用另計 |
|  | 輸卵管攝影檢查 | 確認輸卵管通暢度• 先排子宮鏡，再排輸卵管攝影。• 輸卵管攝影檢查須禁食（含水）4小時。 |  |

## （二） 治療項目

| 時間 | 流程 | 說明 | 備註 |
|------|------|------|------|
| 月經週期第2-3天 | 打排卵針 | 刺激卵泡成長• 打排卵針前，先做陰道超音波+抽血驗(CBC、VDRL、HIV、E2、FSH、LH)。• 進行取卵手術術前準備(1):心電圖檢查、X光檢查。 |  |
| 第3-14天 | 打排卵針 | 刺激卵泡成長• 連續施打排卵針(FSH)8-12天，甚至14天，以刺激卵泡成長。藥物請冷藏保存，不可放於冷凍庫。• 依據醫師指示的時間，回醫院做陰道超音波追蹤卵子的成熟度，約3-4次。 | 因藥物種類、藥物劑型及使用天數不同，費用會有差異 |
| 第15天 | 打破卵針 | 促進卵泡成熟以達到最佳取卵時間• 返院拿破卵針。通常卵泡發育至≥1.8cm為破卵針(hCG)施打的時機。 |  |
